# Filmie
Movie browsing app, made using [themoviedb](https://www.themoviedb.org/) api,
as part of Udacity : Popular Movies (Part-2)

## Add API key
In app/build.gradle replace "COPY YOUR MOVIE DB API KEY HERE" with your API key
